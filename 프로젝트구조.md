# 📁 프로젝트 구조

## 🚀 실행 파일

**Windows:**
```
install-ngrok.bat          # ngrok 설치 (최초 1회)
start-with-ngrok.bat       # 전체 시스템 실행 (메인)
```

**macOS/Linux:**
```
install-ngrok.sh           # ngrok 설치 (최초 1회)
start-with-ngrok.sh        # 전체 시스템 실행 (메인)
```

## 🔧 백엔드 (FastAPI)

```
run.py                    # 서버 실행 파일
requirements.txt          # Python 의존성
.env                      # 환경 변수 (GEMINI_API_KEY)

app/
  ├── __init__.py         # 패키지 초기화
  ├── main.py             # FastAPI 앱 (라우트)
  ├── config.py           # 설정
  ├── models.py           # 데이터 모델
  ├── prompts.py          # AI 프롬프트
  ├── utils.py            # 유틸리티
  └── gemini_client.py    # Gemini API 클라이언트

venv/                     # Python 가상환경
```

## 🎯 핵심 파일

### 실행
- **start-with-ngrok.bat** (Windows) - 한 번에 모든 것 실행
- **start-with-ngrok.sh** (macOS/Linux) - 한 번에 모든 것 실행

### 설정
- **.env** - `GEMINI_API_KEY=your_key_here`
- **mobile/src/services/api.js** - API URL (자동 설정됨)

### 문서
- **실행방법.md** - 사용 방법
- **QUICKSTART.md** - 빠른 시작
